# 데이터베이스

데이터를 저장할 공간

데이터를 지속적으로 저장, 사용하고 보호하는 것

## DBMS

데이터베이스를 관리하는 시스템

- 기능
  - 데이터의 추가/조회/변경/삭제
  - 무결성 유지
  - 트랜잭션 관리
  - 데이터의 백업 및 복원
  - 데이터 보안

DBMS 클라이언트 : 데이터베이스를 사용하는 애플리케이션
