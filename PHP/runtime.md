# PHP runtime

- PHP 런타임은 어플리케이션 코드 및 종속 항목을 설치하고 애플리케이션을 실행하는 소프트웨어 스택입니다.

## PHP 확장 프로그램 사용
- 기본적으로 mbstring, MySQL, CURL등 사용할 수 있지만, APM, AMQP 등 기본적으로 사용되지는 않습니다.


- 다음 PHP 확장 프로그램은 시스템에 사전 설치 및 사용 설정되어 있습니다.
